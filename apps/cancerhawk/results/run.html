Backend

The static site (Vercel/Pages) cannot run Python; it talks to the deployed FastAPI Hermes worker over WebSocket. The worker URL is https://cancerhawk-production.up.railway.app.

python -m app.main   # local dev (port 8765 by default)

Once the backend is reachable, paste your OpenRouter API key, choose models, and run. Hermes publishes completed blocks into results/block-N/, clones the GitHub repo with GITHUB_TOKEN, pushes as hermes-agent, and Vercel auto-rebuilds the public site.
