# solana.new Skills

This repo uses the solana.new agent skill pack.

Install/update locally or on a server:

```bash
curl -fsSL https://www.solana.new/setup.sh | bash
```

That installer populates skill directories for common agent runtimes (`~/.agents/skills`, `~/.codex/skills`, `~/.claude/skills`).

For this repo, use:

```bash
npm run agents:bootstrap
```
