# HyperCinema Cinematography Knowledge Bank

This bank feeds the cinematic worker assembly for prompt jobs, music videos, scene recreations, and token trailers.

## Agent boxes

### Tianshi
- Main agent and final approver.
- Owns world building, cinematography, lighting logic, lensing, movement, and final visual cohesion.
- Rejects repetitive coverage, source drift, accidental debug text, and subtitle-like overlays unless explicitly requested.

### Script Writer
- Extracts the smallest meaningful story from the brief or source.
- Starts on page one.
- Writes scenes late-in and early-out.
- Uses visual storytelling instead of explanatory narration.

### Editor
- Controls shot order, pacing, transitions, and escalation.
- Forces variation in shot size, angle, and movement.
- Removes repetition and blocks default caption behavior.

## Core cinematography rules

- Visual storytelling first. Show emotion, hierarchy, and conflict in the frame before explaining them.
- Maintain spatial logic with clear screen direction, eyelines, and readable geography.
- Use establishing shots to place the audience, mediums to track action, and close-ups for emotional or symbolic locks.
- Use symmetry or one-point perspective when you want ritual, control, obsession, or mythic focus.
- Use low angles for power, high angles for vulnerability, and dutch angles only when instability is the point.
- Motivate camera movement. Tracking shots immerse, push-ins apply pressure, whip pans add urgency, dolly zooms signal psychological distortion.
- Keep lighting logic consistent across coverage. The close-up should still feel like it belongs to the wide shot.
- Use montage when compression is needed. Use long takes when immersion matters more than compression.
- Match cuts, dissolves, hard cuts, and bridge shots must each have a reason.
- No default subtitles, lyric captions, lower thirds, or debug text.

## Editing rules

- Do not repeat the same centered portrait setup in consecutive beats unless repetition itself is the meaning.
- Build escalation through a change in framing, angle, lighting, movement, or transition, not by swapping random pretty images.
- Give the sequence breathing room. Every beat should not peak at the same intensity.
- Protect dialogue clarity and scene rhythm when adapting from a source.

## Source-grounding rules

- If a source URL, embed, or transcript exists, it is a primary reference box, not an ignored attachment.
- Use source title, iconography, rhythm, blocking intent, and emotional promise.
- A transcript is a hidden pacing guide, not an automatic invitation to generate captions.
