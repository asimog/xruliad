# QVAC Integration

QVAC is optional for web deployment and preferred for private/local inference.

Local reference checkout:

- `C:\qvacenterprise`

Runtime integration:

- `packages/qvac`
- `packages/qvac-agent`
- `services/qvac-gateway`

The repo does not vendor QVAC. Use `QVAC_GATEWAY_URL` or `QVAC_BASE_URL` to connect a local OpenAI-compatible QVAC gateway.
