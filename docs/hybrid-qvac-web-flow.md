# Hybrid QVAC/Web Flow

QVAC is preferred for private local reasoning and optional for web usage.

Public tasks can use the cheapest safe cloud route. Private strategy requires QVAC or a local-only path. Wallet/key material is blocked from cloud routing.

Implemented boundaries:

- `packages/qvac`
- `packages/qvac-agent`
- `services/qvac-gateway`
- `packages/privacy`
- `packages/inference-router`
